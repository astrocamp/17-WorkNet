# 17th-WorkNet

## Copy

1. `git clone https://github.com/astrocamp/17th-WorkNet.git` (複製到本地)
2. `cd 17th-WorkNet` (進入專案內容)
3. `git pull origin dev` (拉取最新變更)
4. `git status` (確認狀態)

## Install

1. `poetry shell` (進入虛擬環境)
2. `poetry install` (下載 poetry 相關套件)
3. `npm i` (下載 npm 相關套件)
4. 建立`.env` (SECRET_KEY)

## Npm

- `npm run dev-mac` (mac)
- `npm run dev-win` (win)

## Git Push

1. `poetry run pre-commit run --all-files`
2. `git add <file name>`
3. `cz commit`
4. `git push`

## Git Push (Makefile)

1. `make lint`
2. `git add <file name>`
3. `make commit`
4. `git push`
